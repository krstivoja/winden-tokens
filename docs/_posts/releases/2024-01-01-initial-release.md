🎉 We're thrilled to announce the initial release of **Winden Tokens**!

Winden Tokens is a professional design token management plugin for Figma, designed to help you manage design tokens at scale.

## Key Features

- Spreadsheet-style table for fast bulk editing
- JSON editor for batch updates
- Color palette generation
- Number scales
- Visual graphs
- WCAG contrast checking

Stay tuned for more updates and improvements!
