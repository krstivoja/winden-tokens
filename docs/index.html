Features

Six specialized views

Everything you need to manage design tokens at scale

Table View

Spreadsheet-style editing with inline controls, collection switching, and instant search

  • Edit 100s of variables faster than native UI
  • Move variables between collections via dropdown
  • Auto-groups by slash-separated paths

Shades Generator

Generate 50-950 color palettes with advanced Bezier curve controls

  • Bezier curves for lightness, saturation, hue
  • Managed shades auto-update when base changes
  • Real-time preview with accessibility badges

Steps Generator

Create spacing/sizing scales with Golden Ratio, Major Third, and more

  • 8 preset ratios + custom values
  • Reference existing variables as base
  • T-shirt, numeric, or custom naming

Node Graphs

Interactive graphs for colors and numbers showing all token relationships

  • Drag to create variable aliases
  • Visualize complex token hierarchies
  • Pan, zoom, and keyboard shortcuts

Contrast Checker

WCAG 2.1 compliance with real-time AA/AAA badges

  • Per-variable or per-group checking
  • Pick colors or reference variables
  • Instant ratio calculation

Bulk Editing

CSV and JSON editors for importing/exporting entire token sets

  • Paste from spreadsheets (CSV)
  • JSON editor for power users
  • Preview before applying
